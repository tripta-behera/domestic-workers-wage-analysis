# domestic-workers-wage-analysis
Wage Analysis: Informal Domestic Workers, Delhi

Primary data collected from 16 female migrant domestic workers 
in Jai Hind Camp, Vasant Kunj, New Delhi (2025).

Analysis examines wage determinants using OLS regression in R.
Data collected via structured interviews for 16 households as part of MA Economics 
seminar research, Jawaharlal Nehru University.
